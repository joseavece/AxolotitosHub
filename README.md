# AxolotitosHub 🐟

Repositorio general de nuestro club.  
Aquí pueden subir avances y material de **Electrónica**, **Mecánica** y **Programación**.

## Estructura
- [Electrónica](./Electrónica): esquemas, circuitos y simulaciones.
- [Mecánica](./Mecánica): planos, diseños y documentos mecánicos.
- [Programación](./Programación): código fuente y scripts.

## Cómo contribuir
1. Ve a la carpeta de tu área.
2. Crea una subcarpeta con el nombre de tu proyecto.
3. Agrega dentro:
   - `README.md` → descripción de tu proyecto.
   - Código, esquemas o documentos relacionados.
4. Haz commit con un mensaje claro:  
   `feat(mecánica): agregar planos de estructura del robot`
